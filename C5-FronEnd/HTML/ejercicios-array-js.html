Ejercicios arrays

Ejercicio 1: Manipulación de Arrays Dado el array ["manzana", "banana", "pera", "naranja", "uva"], realiza las siguientes operaciones:

  • Elimina el último elemento y muestra el array resultante.
  • Añade "kiwi" al principio del array.
  • Combina el array con ["fresa", "mango"] y muestra el array combinado.

Ejercicio 2: Filtrado y Mapeo. Dado el array [5, 12, 8, 130, 44], realiza las siguientes operaciones:

  • Filtra los elementos que son mayores de 10 y muestra el array resultante.
  • Mapea los elementos para que sean su cuadrado (elemento * elemento) y muestra el array resultante.
